이어서 작성해줘. 추가로이미지를 마크다운의 ![](링크) 와 같이 바로 출력될 수 있도록 있도록 html을 덧대줘. '코드블록'은 다른 방식으로 표현해줘. 추가로 점검하고 결과물에 무언가를 추가하는등 보완하면서 진행해줘. 특히 아래 기준과 처음 입력의 조건들을 제대로 이행했는지는 무조건 scrutiny의 대상이야. ***full code***를 보내줘. 추가로 읽는데 걸리는 시간이랑 급할때 꼭 읽어야 하는 부분은 처음 단락에 같이 적어줘. 그리고 아래 형식에 맞춰서 '디자인'도 수준급으로 진행해줘.

**⭐ 이미지 생성 규칙 (2026-01-09):**
• 실사 이미지만 사용 (일러스트/다이어그램 금지)
• 텍스트 라벨 = 디지털 손글씨 스타일
• GitHub raw URL 필수 (Unsplash 금지)

이미지 Alt 태그 확인: * HTML 내에 이미지 태그를 넣으실 때 <img src="https://raw.githubusercontent.com/..." alt="신화 전설 민담 차이 비교표"> 처럼 alt 속성을 꼭 넣어주세요. 네이버 검색 엔진이 이미지를 이해하는 데 큰 도움이 됩니다.

모바일 <-> pc 모두 호환되게 해줘. 또한 

No CSS Block: <style> 태그를 절대 사용하지 말고, 모든 스타일은 태그 내 style="" 속성(인라인)으로만 작성할 것.

No Classes: class나 id 속성을 사용하지 말 것. 오직 인라인 스타일로만 디자인할 것.

Box to Table: 박스(Box) 디자인이 필요하면 div 대신 **1x1 표(table)**를 사용할 것. (네이버는 표 내부의 배경색과 테두리를 가장 잘 보존함)

RGB Colors: 색상 값은 가능한 rgb(0, 0, 0) 형식으로 표현할 것.

Line Height: 가독성을 위해 모든 <p> 태그에 line-height: 1.8;을 명시할 것.

즉, 네이버 블로그용 인라인 스타일(Inline Style)로 변환해줘.
